# Portfolio

This is a [Next.js](https://nextjs.org/) project.

## Getting Started

First, install node_modules:

```bash
npm i
# or
yarn install
# or
pnpm i
# or
bun i
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
